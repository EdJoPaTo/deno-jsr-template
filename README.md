## Install

Node.js:

```bash
npm install deno-node-module-template
```

Deno:

```ts
import {/* ... */} from "https://deno.land/x/deno_node_module_template/mod.ts";
```

## Usage

```ts
import { myFunction } from "deno-node-module-template";

myFunction("unicorns");
//=> 'unicorns & rainbows'
```
